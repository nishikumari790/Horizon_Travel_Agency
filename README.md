# ✈️ HorizonTravel Travel Agency Website

A fully responsive, professional travel agency website built with **Bootstrap 5**, custom CSS, and vanilla JavaScript.

## 📁 Project Structure

```
travel-agency/
├── index.html          → Home Page
├── about.html          → About Us Page
├── destinations.html   → Destinations Page (with filter + gallery)
├── packages.html       → Packages Page (with pricing tiers)
├── contact.html        → Contact Us Page (with validated form + FAQ)
├── css/
│   └── style.css       → Custom styles (CSS variables, components, animations)
├── js/
│   └── main.js         → JavaScript (filter, validation, counter, lightbox, scroll)
└── README.md           → This file
```

## 🚀 Features

### Pages
| Page | Features |
|------|----------|
| **Home** | Hero with search widget, Why Us cards, Top Destinations, Featured Packages, Animated Counters, Testimonials, CTA |
| **About Us** | Story section, Mission & Values, Team profiles, Awards & Certifications |
| **Destinations** | Live search bar, Category filter (All / Asia / Europe / Africa / Americas / Oceania), 9 destination cards, Photo gallery with lightbox |
| **Packages** | Category filter, 6 full packages with features & pricing, 3-tier pricing comparison (Explorer / Premium / Ultra Luxury) |
| **Contact Us** | Full validated contact form, Contact info card, FAQ accordion, Map placeholder |

### Technical Features
- ✅ **Fully Responsive** — Mobile, Tablet, Desktop (Bootstrap 5 grid)
- ✅ **Bootstrap 5.3** — Components, accordion, collapse, grid
- ✅ **Font Awesome 6** — Icons throughout
- ✅ **Google Fonts** — Playfair Display + DM Sans
- ✅ **Fixed Navbar** — Scroll effect + active page highlight
- ✅ **Destination Filter** — JavaScript category filtering
- ✅ **Live Search** — Real-time destination search
- ✅ **Animated Counters** — IntersectionObserver-triggered
- ✅ **Contact Form Validation** — Client-side with visual feedback
- ✅ **Photo Lightbox** — Click to view full size in gallery
- ✅ **Back to Top Button** — Auto-show/hide on scroll
- ✅ **Scroll Animations** — Fade-in-up on scroll
- ✅ **Newsletter signup** — Footer on all pages

## 📦 How to Run Locally

1. **Clone or download** the project folder
2. Open `index.html` in any modern browser
3. No build step required — pure HTML/CSS/JS

---

### GitHub Pages
1. Push project to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Site will be live at `https://yourusername.github.io/repo-name`
